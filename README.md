## プロジェクトの初期構築方法
1. プロジェクトのクローン：まず、GitHubリポジトリからプロジェクトをクローンします。
```
git clone https://github.com/Ryota-Nakazono/techgeek-backend-tutorial.git
```

2. 依存関係のインストール：yarn installコマンドを使って、package.jsonにリストされた依存関係をインストールします。
```
cd repository
yarn install
```

3. アプリケーションの実行
```
yarn dev
```
